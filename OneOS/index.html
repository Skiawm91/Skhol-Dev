Sakura OneOS 簡介

發布於2023年2月12日，首個版本搭載純文字GUI，
之後的版本與按鈕組件一同使用。
支援 Windows XP 及更新版本，並且搭配 SakuraPC 使用。
不過，更新系統需要 Git 才可運作，
3.00.1011 開始則內建 Wget 64位元版本 (3.00.1259開始使用32位元版本)。

【注意】32位元作業系統僅能使用 3.00.1259 (及更新版本) 的更新系統。
