【新聞】新的 SdoneOS 技術一覽！

2025/3/7 - Skiawm91

據了解，在 (Skhol Dev) OneOS 結束開發後，
開發者開始製作新的系統，名 (Skhol Dev) SdoneOS，
從最新的圖片來看，SdoneOS 似乎不具備圖形化介面，
這有可能是因為OneOS使用了圖形化後變得複雜的代碼，
使開發者非常不便，現在的方式可以減少開發者的不便。 使開發者不再願意為新系統加入圖形化介面。
新的系統介面不採用圖形化，而是使用 Command Line。
接著，新的系統有許多的套件 (額外安裝)，包括語言套件，
使用 SDPKGM (Skhol Dev Package Manager) 來管理。
而新系統也將採模組化來設計，舉例來說：
- sdone.bat (系統本體)
- sdlogin.bat (登入介面)
- bash.bat (Shell)
在舊的OneOS中，這些東西皆在 Kernel32.bat 中，
使開發者非常不便，現在的方式可以減少開發者的不便。
